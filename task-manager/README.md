# Task Manager (super simple)

A lightweight task tracker with add, edit, delete, and completion features

## Tools I used
- Vue 3
- Vite
- Pinia

## How to install and run
- clone this repo
- `npm install`
- `npm run dev`
- open the link in terminal (usually http://localhost:5173)

## How to build
- `npm run build`
- it makes a dist folder
