# Task Manager (super simple)

Just a tiny task app, I am still learning so it is not perfect.

## Tools I used
- Vue 3
- Vite
- Pinia

## How to install and run
- clone this repo
- `npm install`
- `npm run dev`
- open the link in terminal (usually http://localhost:5173)

## How to build
- `npm run build`
- it makes a dist folder
